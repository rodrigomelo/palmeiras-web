Palmeiras Agenda para web, iOS e Android

Última atualização: 5 de julho de 2026

O Palmeiras Agenda é uma agenda esportiva independente para acompanhar jogos, tabela, notícias, lembretes e calendário. Salvo autorização formal documentada, ele não é o aplicativo oficial da Sociedade Esportiva Palmeiras e não representa afiliação, patrocínio ou endosso do clube.

Como pedir suporte

Para relatar erro, pedir ajuda, informar problema de privacidade ou solicitar análise relacionada a dados, abra uma solicitação no repositório público do projeto.

Abrir solicitação Ver solicitações

Problemas comuns

  • Notificações: verifique se o sistema permitiu notificações para o app.
  • Calendário: autorize o calendário apenas se quiser criar eventos de partidas no aparelho.
  • Dados de jogos: horários, tabela e notícias dependem das fontes públicas e do serviço de API do projeto.
  • Links externos: notícias abrem sites de terceiros e ficam sujeitas às políticas desses sites.

Privacidade e dados

O app não cria conta e não envia dados pessoais do usuário para o servidor no produto inicial. Lembretes e eventos de calendário criados no aparelho podem ser removidos pelo próprio sistema operacional. A política completa está em palmeiras.rodrigolanna.com.br/privacy.html.
